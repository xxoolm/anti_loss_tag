## 文档说明

### 文档组织结构
```
docs/
├── index.md          # 本文档（导航入口）
├── user/             # 用户文档（使用和配置）
├── technical/        # 技术文档（开发实现）
├── reference/        # 参考文档（硬件和标准）
├── java/             # Java参考（学习借鉴）
├── archive/          # 归档区（历史报告）
└── old_docs/         # 旧版本文档（已保留）
```

### 内容来源说明
- **新版本文档**：基于旧版本文档重组和优化，消除重复，结构更清晰
- **旧版本文档**：保留在 `docs/old_docs/` 目录，可参考原始内容
- **归档报告**：历史修正和完成报告保留在 `docs/archive/reports/`

### 如何使用本文档
1. **首次使用**：从 [快速开始](user/getting-started.md) 开始
2. **遇到问题**：查看 [故障排除](user/troubleshooting.md)
3. **深入了解**：阅读 [技术文档](technical/)
4. **硬件开发**：参考 [硬件文档](reference/)
