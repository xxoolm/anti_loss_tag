## 文档说明

### 文档组织结构
```
docs/
├── 索引.md            # 本文档（导航入口）
├── 用户文档/          # 用户文档（使用和配置）
├── 技术文档/          # 技术文档（开发实现）
├── 参考资料/          # 参考文档（硬件和标准）
├── Java参考/          # Java参考（学习借鉴）
├── 旧文档/            # 旧版本文档（已保留）
└── 归档/              # 归档区（历史报告）
```

### 内容来源说明
- **新版本文档**：基于旧版本文档重组和优化，消除重复，结构更清晰
- **旧版本文档**：保留在 `docs/旧文档/` 目录，可参考原始内容
- **归档报告**：历史修正和完成报告保留在 `docs/归档/报告/`

### 如何使用本文档
1. **首次使用**：从 [快速开始](用户文档/快速开始.md) 开始

3. **深入了解**：阅读 [技术文档](技术文档/)
