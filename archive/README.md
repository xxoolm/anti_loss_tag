# 归档目录说明

本目录包含项目开发过程中的参考文档、旧版本代码和临时文件。

## 目录结构

### old_versions/
- `anti_loss_tag_v1/` - 第一版集成代码
- `anti_loss_tag_v2/` - 第二版集成代码（部分改进）

### reference_docs/
- `KT6368A.pdf` - KT6368A芯片完整数据手册
- `KT6368A_V1.0.pdf` - KT6368A V1.0版本数据手册
- `KT6368A 定制固件 通用 Python 开发文档.md` - Python开发指南
- `KT6368A 硬件开发文档（SOP-8）.md` - 硬件设计指南

### temp_files/
- `MyApplication.java` 和 `MyApplication$3.java` - Android示例代码
- `init_git.sh` - Git仓库初始化脚本
- `anti_loss_tag_optimized_v2.zip` - 临时备份文件

## 说明

这些文件仅供参考和追溯历史，不是项目运行所必需的。

当前活跃的集成代码位于 `custom_components/anti_loss_tag/` 目录。
